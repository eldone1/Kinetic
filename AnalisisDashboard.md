Necesito diseñar dashboards modernos, profesionales y funcionales para un sistema web de gestión de una clínica de rehabilitación física.

La interfaz debe estar pensada para uso diario, con gran cantidad de información, navegación rápida y enfoque en productividad. Debe ser responsive, limpia y con jerarquía visual clara.

Requisitos de diseño
Diseño tipo SaaS moderno.
Inspirado en Stripe, Linear, Notion, Clerk, Vercel y sistemas médicos premium.
Uso de tarjetas KPI.
Gráficos claros y profesionales.
Tablas modernas con filtros.
Colores suaves orientados al sector salud.
Excelente experiencia para escritorio.
Diseñar wireframe detallado de cada dashboard.
Explicar la distribución visual de cada sección y justificar por qué cada métrica es importante para ese rol.
